```md
# Troubleshooting

## Camera Not Detected
- Check camera cable
- Enable camera in `raspi-config`
- Test with `libcamera-still`

## Low FPS
- Reduce frame resolution
- Disable GUI
- Use Raspberry Pi 4

## False Detections
- Improve lighting
- Adjust HSV values
- Reduce background noise

## Motors Not Moving
- Check power supply
- Confirm GPIO pin mapping
- Test motor driver separately
````
